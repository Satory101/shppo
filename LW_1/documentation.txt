================================================================================
                                DocumentCommand
================================================================================
    Описание: Интерфейс для команд, работающих с документами. Определяет методы
              для выполнения и отмены команд.
    Методы:
        execute():   Выполняет команду.
        undo():      Отменяет выполненную команду.
================================================================================
                                ChangeTextCommand
================================================================================
    Описание: Команда для изменения текста в документе. Используется в паттерне
              "Команда".
    Поля:
        textDocument:   Экземпляр TextDocument, с которым работает команда.
        newText:        Новый текст для установки в документ.
        previousText:   Текст документа до внесения изменений.

    Методы:
        execute():   Выполняет изменение текста документа, сохраняя предыдущий текст.
        undo():      Отменяет последнее изменение текста документа, восстанавливая
                     предыдущий текст.
================================================================================
                                DocumentFactory
================================================================================
    Описание: Интерфейс для фабрики документов. Определяет метод для создания
              документа заданного типа.
    Методы:
        createDocument(String type):   Создает и возвращает документ указанного
                                       типа.
================================================================================
                                ConcreteDocumentFactory
================================================================================
    Описание: Конкретная фабрика для создания документов различных типов.
              Реализует интерфейс DocumentFactory.
    Методы:
        createDocument(String type):   Создает и возвращает документ указанного
                                       типа. Поддерживаемые типы: "TextDocument",
                                       "SpreadsheetDocument".
================================================================================
                                DocumentObserver
================================================================================
    Описание: Интерфейс для наблюдателей за изменениями документа. Определяет
              метод для обновления наблюдателя.
    Методы:
        update(Document document):   Вызывается при обновлении документа для
                                     уведомления наблюдателя.
================================================================================
                                ConcreteDocumentObserver
================================================================================
    Описание: Конкретный наблюдатель, который сохраняет документ при его
              обновлении. Реализует интерфейс DocumentObserver.
    Поля:
        saveCommand:   Экземпляр SaveTextDocumentCommand, используемый для
                       сохранения документа.
    Методы:
        update(Document document):   Вызывается при обновлении документа. Выводит
                                     сообщение об обновлении и выполняет команду
                                     сохранения.
================================================================================
                                Document
================================================================================
    Описание: Абстрактный класс для документов. Поддерживает шаблоны "Наблюдатель"
              и "Снимок" для управления состояниями и уведомлениями.
    Поля:
        observer:         Наблюдатель за изменениями документа.
        historyLogger:    Логгер истории состояний документа.

    Методы:
        getContent():                             Возвращает текущее содержимое
                                                  документа.
        setContent(String content):               Устанавливает новое содержимое
                                                  документа и уведомляет
                                                  наблюдателя.
        createMemento():                          Создает и возвращает снимок
                                                  текущего состояния документа.
        restoreFromMemento(DocumentMemento m):    Восстанавливает состояние
                                                  документа из снимка.
        addObserver(DocumentObserver observer):   Добавляет наблюдателя за
                                                  изменениями документа.
        removeObserver():                         Удаляет наблюдателя.
        undoLastChange():                         Выполняет откат к последнему
                                                  сохраненному состоянию.
        notifyObserver():                         Уведомляет наблюдателя об
                                                  изменениях.
        saveStateToHistory():                     Сохраняет текущее состояние
                                                  документа в историю.
================================================================================
                                TextDocument
================================================================================
    Описание: Класс текстового документа. Реализует методы получения и изменения
              содержимого, а также создания и восстановления снимков состояния.
    Поля:
        content:   Содержимое текстового документа.

    Методы:
        getContent():                             Возвращает текущее содержимое
                                                  документа.
        setContent(String content):               Устанавливает новое содержимое
                                                  документа и сохраняет
                                                  состояние в историю.
        createMemento():                          Создает и возвращает снимок
                                                  текущего состояния документа.
        restoreFromMemento(DocumentMemento m):    Восстанавливает состояние
                                                  документа из снимка.
================================================================================
                                DocumentMemento
================================================================================
    Описание: Абстрактный класс для снимков состояния документа. Содержит тип
              документа.
    Поля:
        type:   Тип документа.

    Методы:
        getType():   Возвращает тип документа.
================================================================================
                                TextDocumentMemento
================================================================================
    Описание: Снимок состояния текстового документа. Сохраняет текст документа
              на момент создания снимка.
    Поля:
        text:   Текст документа.

    Методы:
        getText():   Возвращает сохраненный текст документа.
================================================================================
                                DocumentHistoryLogger
================================================================================
    Описание: Класс для ведения истории состояний документа. Используется для
              сохранения и отката состояний.
    Поля:
        history:   Стек для хранения снимков состояний документа.

    Методы:
        addMemento(DocumentMemento m):   Добавляет снимок состояния в историю.
        undo():                          Выполняет откат к предыдущему состоянию
                                         и возвращает его снимок.
        getHistory():                    Возвращает стек истории состояний.
================================================================================
                                SaveTextDocumentCommand
================================================================================
    Описание: Команда для сохранения текущего состояния текстового документа в
              историю. Используется в паттерне "Команда".
    Поля:
        document:        Экземпляр TextDocument для сохранения.
        historyLogger:   Экземпляр DocumentHistoryLogger для ведения истории.

    Методы:
        execute():   Сохраняет текущее состояние документа в историю.
        undo():      Выполняет откат к предыдущему сохраненному состоянию.
================================================================================
                                ShowTextDocumentCommand
================================================================================
    Описание: Команда для отображения содержимого текстового документа в
              отдельном окне. Используется в паттерне "Команда".
    Поля:
        document:   Экземпляр TextDocument для отображения.

    Методы:
        execute():   Открывает окно с текущим содержимым документа.
        undo():      (Не используется в этой команде)
================================================================================
                                DocumentEditorGUI
================================================================================
    Описание: Графический интерфейс для редактирования текстового документа.
              Поддерживает внесение изменений, откат и просмотр истории.
    Поля:
        document:        Экземпляр TextDocument, используемый в редакторе.
        historyLogger:   Экземпляр DocumentHistoryLogger для ведения истории.
        saveCommand:     Экземпляр SaveTextDocumentCommand для сохранения
                         состояния документа.

    Методы:
        (Конструктор):   Инициализирует графический интерфейс и настраивает
                         обработчики событий.
================================================================================
                                Main
================================================================================
    Описание: Класс для запуска приложения редактора документа.
    Методы:
        main(String[] args):   Создает и отображает экземпляр DocumentEditorGUI.
================================================================================
